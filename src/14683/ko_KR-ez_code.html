전기로 정확하게

문제

당신은 (x축과 평행하게) 동서로 뻗은 정수 번호의 도로와 (y축과 평행하게) 남북으로 뻗은 정수 번호의 거리로 이루어진 격자 도시에 살고 잇습니다. 도로와 거리의 길이는 무한하고 모든 x좌표에는 해당하는 거리가, 모든 y좌표에는 해당하는 도로가 있습니다. 모든 교차점에는 정수 좌표가 매겨져 있습니다: 예를 들어, 7번 거리와 -3번 도로는 (7, -3)에서 교차합니다.

당신은 인접한 교차점으로 이동할 때 전기 배터리 1을 사용하는 특별한 전기 차를 타고 있습니다: 인접한 교차점으로 이동한다는 것은, 북쪽 또는 남쪽으로 가 다음 도로로 이동하거나 동쪽 또는 서쪽으로 가 다음 거리로 이동하는 것을 말합니다. 배터리가 닳을 때까지 각 교차점에서 당신의 차는 좌회전하거나, 우회전하거나, 직진하거나, 유턴합니다. 여행 중 같은 교차점을 여러 번 들를 수도 있습니다.

당신은 출발하는 교차점, 도착하는 교차점, 그리고 남아 있는 전기 배터리의 양을 알고 있다고 가정합시다. 도착점에 배터리가 0이 된 상태로 도착하도록 출발점에서 도착 교차점까지 가지고 있는 전기 배터리의 양을 이용해 여행할 수 있는지 판단합시다.

입력

입력은 세 줄로 주어집니다. 첫 줄은 $a$에 $b$가 이어서 주어지는데, 출발 좌표 ($a$, $b$)를 나타냅니다. ($-1\,000 \le a \le 1\,000$; $-1\,000 \le b \le 1\,000$).

둘째 줄은 $c$에 $d$가 이어서 주어지는데, 도착 좌표 ($c$, $d$)를 나타냅니다. ($-1\,000 \le c \le 1\,000$, $-1\,000 \le d \le 1\,000$)

셋째 줄에는 출발할 때 배터리에 남아있는 전기 배터리의 양을 의미하는 정수 $t$가 주어집니다. ($0 \le t \le 10\,000$)

[역자 - 대회에서는 부분 점수가 있었던 듯 합니다. 백준에서는 없으므로 관련 설명은 생략합니다.]

출력

정확히 $t$의 전기 배터리를 사용해 시작 좌표에서 도착 좌표로 이동할 수 있다면 Y를 출력합니다. 그렇지 않을 경우 N을 출력합니다.

힌트

예제 입력 1에 대한 설명

가능한 경로로 (3, 4) - (4, 4) - (4, 3) - (3, 3)가 있습니다.

예제 입력 2에 대한 설명

정확히 배터리 2를 사용하여 (10, 2) - (10, 4)로 이동할 수 있습니다.

다음 경로를 통해 정확히 배터리 4를 사용하여 이동할 수 있습니다:

(10, 2) → (10, 3) → (11, 3) → (11, 4) → (10, 4).

정확히 배터리 5를 사용하여 (10, 2) - (11, 4)로 이동할 수 있습니다.

(10, 2) → (10, 3) → (11, 3) → (12, 3) → (12, 4) → (11, 4).

하지만, 정확히 배터리 5를 사용하여 (10, 2)에서 (10, 4)로 이동하는 경로는 존재하지 않습니다.
