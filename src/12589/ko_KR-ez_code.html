똑딱이 사슬 (Small)

문제

똑딱이는 한쪽에 출력 소켓에 연결할 수 있도록 입력 플러그가 있고, 반대쪽에 전구나 다른 기기에 연결하기 위한 출력 소켓이 있는 작고 유용한 장치입니다.

똑딱이가 켜진 상태면 입력 플러그에서 전력을 받게 되고, 장치의 출력 소켓에 연결된 장치 역시 전력을 받습니다. 당신이 손가락을 튕겨 -- 똑딱 소리를 내면 -- 그 순간 전력을 받고 있는 모든 똑딱이는 켜진 상태와 꺼진 상태를 바꿀 것입니다.

괴짜 같은 아이디어로 우주를 파괴하기 위해, 당신은 N 개의 똑딱이 장치를 구매한 후 첫 번째 장치를 전력 장치에 연결하고, 두 번째 장치를 첫 번째 장치에 연결하는 방식으로 사슬을 만들었습니다. 전구는 N 번째 똑딱이에 연결되어 있습니다.

처음에 모든 똑딱이는 꺼져 있으므로 첫 번째 장치만이 소켓에서 전력을 받고 불은 꺼져 있습니다. 당신이 손을 한 번 튕기면, 첫 번째 똑딱이가 켜지며 두 번째 장치에 전력을 공급합니다. 다시 손가락을 튕기면 두 똑딱이의 상태가 바뀌며, 즉시 두 번째 장치는 켜진 상태가 되지만 전력은 끊깁니다. 세 번째로 손을 튕기면, 첫 번째 똑딱이의 상태가 바뀌며 두 번째 장치에 전력을 공급합니다. 이제 두 똑딱이가 모두 켜진 상태이므로, 전구가 두 번째 똑딱이에 연결되어 있다면 불이 켜질 것입니다.

당신은 이 행위를 몇 시간 동안 합니다. K번 손가락을 튕긴 후 불은 켜진 상태입니까, 꺼진 상태입니까? 전구는 전구가 꽂힌 똑딱이에 전력이 공급될 때만 켜집니다.

입력

입력 첫 줄에 테스트 케이스의 수 T 가 주어집니다. 이어 T개의 줄이 주어집니다. 각 줄은 두 정수 N 과 K 로 이루어져 있습니다.

제한

  • 1 ≤ T ≤ 10,000.
  • 1 ≤ N ≤ 10;
  • 0 ≤ K ≤ 100;

출력

각 테스트 케이스에 대해 x는 (1부터 시작하는) 테스트 케이스의 번호이고 y는 "ON" 또는 "OFF"인 전구의 상태인 한 줄 "Case #x: y"를 출력합시다.
