숫제

문제

아프가니스탄의 아름다운 마을에서 두 자매가 숙제를 대신 해줄 간단한 게임 프로그램을 짜려고 합니다. 숙제는 두 수의 합과 곱을 구하는 것입니다. 자매를 도와 이를 구해주는 프로그램을 작성해 봅시다.

입력

첫째 줄에 테스트 케이스의 수 T가 주어집니다.

각 테스트 케이스의 첫째 줄에 문제의 수 N이 주어집니다. 다음 N 개의 줄에 합과 곱을 구할 두 정수 Ai와 Bi가 주어집니다.

출력

Ai와 Bi가 주어질 때마다 한 줄에 두 수의 합과 곱을 차례로 출력합니다.

제한

  • 1 ≤ T ≤ 10
  • 1 ≤ N ≤ 50
  • 0 ≤ Ai, Bi ≤ 20,000
