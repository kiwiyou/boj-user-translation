Remainder

문제

Does (A+B)%C equal to ((A%C) + (B%C))%C?

Does (A×B)%C equal to ((A%C) × (B%C))%C?

Given three integers A, B, and C, calculate the four values above.

Input

Three integers A, B, and C are given on the first line. (2 ≤ A, B, C ≤ 10000)

Output

Print four lines, each containing (A+B)%C, ((A%C) + (B%C))%C, (A×B)%C, and ((A%C) × (B%C))%C, respectively.
