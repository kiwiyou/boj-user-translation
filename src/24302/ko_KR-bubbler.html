빵 배달

문제

구사과네 빵집은 두 개의 배달 회사를 통해 빵을 배달해 준다. 거리에 따른 각 배달 회사의 배달료는 다음과 같다. 아래에서 x는 배달 거리를 km 단위로 썼을 때 정수 부분이다.

표 1. 배달 회사 1의 배달료

배달 거리 배달료
x ≤ 5 km 4달러
5 km < x ≤ 10 km 7달러
10 km < x ≤ 20 km 12달러
20 km < x ≤ 30 km 17달러
30 km < x 정수 km당 57센트

표 2. 배달 회사 2의 배달료

배달 거리 배달료
x ≤ 2 km 기본 90센트 + 정수 km당 90센트
2 km < x ≤ 5 km 기본 1달러 + 정수 km당 85센트
5 km < x ≤ 20 km 기본 1달러 25센트 + 정수 km당 80센트
20 km < x ≤ 40 km 기본 3달러 25센트 + 정수 km당 70센트
40 km < x 기본 9달러 25센트 + 정수 km당 55센트

구사과는 두 개의 배달 주문을 받았다. 각각의 주문에 대해 배달료를 적게 내는 배달 회사를 선택했을 때, 총 배달료를 출력하시오.

입력

두 주문의 배달 거리가 한 줄에 공백을 사이에 두고 주어진다. 배달 거리의 단위는 미터(m)이며, 모두 정수이다.

출력

총 배달료를 "달러.센트"의 형식으로 출력한다.

제한

  • 100 ≤ x ≤ 100 000
