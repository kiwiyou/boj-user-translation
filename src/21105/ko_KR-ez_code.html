새해

문제

나오미는 작년에 도넛 매출을 확인했습니다. 도넛 맛이 다양하기에, 몇몇 도넛은 다른 도넛보다 더 많이 팔립니다. 내년 매출을 극대화하기 위해, 나오미는 각 도넛 맛마다 정보를 확인합니다. 정확히는 $N$개 맛의 정보를 확인합니다. 각 맛에 대해 나오미는 원래 가격 $O$, 새 가격 $P$, 가격 변화 $C$를 확인합니다. 가격 변화는 공식 $C = 100% * (P - O) / O$를 통해 계산합니다.

안타깝게도 지난 며칠 밤 중 한 번, 나오미는 데이터를 분석하다가 각 도넛 맛의 원래 가격이 적힌 페이지에 통째로 커피를 흘렸습니다.

나오미가 각 도넛 맛의 원래 가격 $O$를 복원하도록 도와줄 수 있나요?

입력

첫 줄에 도넛 맛의 수 $N$이 주어집니다. ($1 \le N \le 10 ^ 4$)

다음 $N$개의 줄은 각 줄마다 하나의 도넛 맛을 설명하는데, 소수 두 자리 수인 두 실수 $P$와 $C$가 주어집니다. ($1 \le P \le 1\,000.00, -1\,000.00 \le C \le 1\,000.00$)

출력

$N$개의 줄을 출력합니다. $i$번째 줄에 $i$번째 도넛 맛의 원래 가격 $O$를 절대 오차 또는 상대 오차 $10 ^ {-5}$ 이내로 출력합니다.
