이진수

문제

양의 정수 $n$이 주어질 때, $n$을 이진법으로 표현했을 때 1이 오는 위치를 찾읍시다. 가장 적은 값을 나타내는 비트의 위치는 0입니다.

아래의 프로그램을 작성합시다:

  • 표준 입력으로 양의 정수 $n$을 받는다.
  • $n$을 이진수로 표기했을 때 1이 나오는 위치를 계산한다.
  • 결과를 표준 출력으로 작성한다.

입력

유일한 줄에 $1 \le n \le 10 ^ 6$을 만족하는 하나의 정수 $n$이 주어집니다.

출력

출력은 정수의 증가 수열이 하나의 공백으로 구분된 모양이어야 합니다 - 이 수열은 입력의 숫자를 이진수로 표기했을 때 1이 나오는 위치입니다.
