가뭄

문제

배열의 인접한 두 원소를 선택해 각각 $1$씩 감소시키는 시행을 반복하여 모든 원소가 같은 값을 가지도록 만들 수 있다면 "좋은 배열"이라고 하자. 단, 모든 원소는 항상 음이 아닌 정수여야 한다.

배열 $H=[H_1,H_2,\ldots,H_N]$이 주어질 때, $h_i\le H_i$를 만족하는 모든 배열 $h=[h_1,h_2,\ldots,h_N]$ 중에서 "좋은 배열"의 개수를 구하라. 답이 커질 수 있으니 $10^9+7$로 나눈 나머지를 출력하라.

입력

첫 번째 줄에는 $N$이 주어진다.

두 번째 줄에는 $H_1,H_2,\ldots,H_N$이 공백으로 나뉘어 주어진다.

출력

$h_i\le H_i$를 만족하는 좋은 배열 $h$의 개수를 $10^9+7$로 나눈 나머지를 출력하라.

$h_i\le H_i$를 만족하는 좋은 배열 $h$가 $(9+1)\cdot (11+1)\cdot (7+1)$개 존재한다.

$h=[8,10,5]$일 때, $h_2$와 $h_3$을 각각 $1$씩 두 번 감소시키고 $h_1$과 $h_2$를 각각 $1$씩 다섯 번 감소시키면 모든 원소가 같은 값 $3$을 가지게 된다.

$h=[0,1,0]$일 때, 모든 원소가 같은 값을 가지도록 만들 수 없다.
