짱세트

문제

짱세트는 세트 게임을 약간 바꾼 게임이다. 세트 게임은 81 개의 카드를 가지고 진행하며, 각 카드는 같은 모양이 1 개부터 3 개까지 그려져 있다.

화살 모양 마름모: 타원:

한 카드에 그려진 모양은 빨간색, 녹색, 파란색 중 하나로 칠해져 있다. (각각 R, G, B) 그리고 다음 무늬 중 하나가 그려져 있다.

무늬 없음: 세로줄: 그물:

그러면 총 81 가지의 가능한 조합이 만들어진다. 카드 세 장을 뽑아 네 가지 속성 (개수, 색, 무늬, 모양) 각각이 세 카드 모두에서 같거나, 세 카드 모두에서 서로 다른 경우에 세트를 만들 수 있다. 예를 들어 다음 1, 2, 3 번 카드는 세트를 이룬다. (개수가 서로 다르고, 색이 같고, 무늬가 서로 다르고, 모양이 서로 다르다) 반면 1, 2, 4 번 카드는 세트를 이루지 않는다. (여러 이유가 있지만 1 번과 4 번이 같은 개수이고, 2 는 다른 개수를 가진다.)

두 카드가 주어졌을 때, 세트를 이루는 나머지 한 카드는 유일하다.

세트 게임의 카드는 {개수}{색}{무늬}{모양} 형식으로 표현된다. 예를 들어 위 네 개의 카드는 순서대로 2GSD, 3GFA, 1GEO, 2BSA이다. {무늬}는 E (무늬 없음), S (세로줄) or F (그물) 중 하나이다. {모양}은 A (화살 모양), D (마름모), O (타원) 중 하나이다.

짱세트 카드는 2GSD와 세트를 이루면서 2GSD가 아닌 서로 다른 두 카드를 90 도 회전해 위아래로 이어붙인 형태이다. 위 그림에서는 3GFA와 1GEO가 해당된다.

짱세트 카드 세 장을 골라 위쪽 세 장의 세트 카드가 세트를 이루고 아래 세 장의 세트 카드도 세트를 이룬다면 (카드를 뒤집어도 된다) 짱세트를 이룬다고 말한다.

예를 들어 아래 그림에서 짱세트는 { 1, 뒤집은 2, 3 }, { 1, 2, 4 }, { 뒤집은 1, 3, 4 }, { 2, 3 , 뒤집은 4 }의 네 가지이다.

서로 다른 짱세트 카드들을 입력으로 받아 만들 수 있는 짱세트의 가짓수를 출력하여라.

입력

입력은 여러 줄로 주어진다. 첫째 줄에는 짱세트 카드의 개수 N이 주어진다. (4 ≤ N ≤ 20) 다음 N 개 줄에는 각 줄마다 짱세트 카드의 위쪽 세트 카드를 나타내는 네 자리 문자열과 아래쪽 세트 카드를 나타내는 네 자리 문자열이 주어진다.

출력

첫째 줄에 가능한 짱세트의 가짓수를 출력한다.
