최대 해밍 거리

문제

$n$개의 $2^m$보다 작은 음이 아닌 정수 $a_1$, $a_2$, $\cdots$, $a_n$이 있다. 여러분은 배열 $a$의 각 원소마다 다른 원소와의 최대 해밍 거리를 계산해야 한다.

두 음이 아닌 정수의 해밍 거리는 두 수를 이진법으로 나타내었을 때 서로 다른 자리의 수로 정의한다. 필요하다면 한 수의 앞쪽에 $0$을 붙여 자릿수를 같게 만든다.

엄밀하게는 각 $i$마다 다음 값을 계산해야 한다. $$\max_{1 \le j \le n}hamming(a_i, a_j)$$

입력

첫째 줄에 두 정수 $n$과 $m$이 주어진다. $(1 \le n \le 2^m$, $1 \le m \le 20)$

둘째 줄에 $n$개의 정수 $a_i$가 주어진다. $(0 \le a_i \lt 2^m)$

출력

한 줄에 $n$개의 정수를 공백으로 구분하여 출력한다. $i$번째 수는 $a_i$와 $a$의 다른 원소와의 최대 해밍 거리이다.

서브태스크

번호배점제한
120

$m \le 10$

225

$m \le 16$

325

추가적인 제약 조건이 없다.

힌트

세 번째 예제를 보자. $3$, $4$, $6$, $10$은 각각 이진법으로 $0011$, $0100$, $0110$, $1010$이다. $3$과 $4$는 $3$개 자리가 다르고, $4$와 $10$ 또한 마찬가지다. 한편 $6$은 다른 어떤 수와도 $2$개 자리만 다르다.
