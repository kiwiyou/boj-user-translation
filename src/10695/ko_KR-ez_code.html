달리아

문제

ICPC 커뮤니티는 전세계를 잇는 하나의 가족 같은 느낌입니다; 대회 주최자, 심판, 코치가 대회 장소에 가족을 데려와 대회 진행 중 아이들을 쉽게 볼 때마다 이 느낌은 더 강해집니다. 2013년 ICPC에는 (ACPC 자금 조달 팀 소속이자 1998년부터 페글라의 절친인) 모하메드 달리아 엘-헤프니의 어린 아들이 왔습니다. IBM 휴식 공간을 배회하면서 아이는 체스를 하는 사람에게 어떻게 하는 것인지 물어봤고, 한 명이 체스 입문 책을 아이에게 주었습니다.

책의 일부분은 나이트의 복잡한 움직임에 대해 설명합니다. 독자를 가르치기 위해 책은 많은 페이지를 나이트를 엄청 큰 보드에 올려놓고 도착 위치 하나를 표시한 다음, 한 번의 움직임으로 나이트가 현재 위치에서 도착 위치에 도달할 수 있는지 물어보는 데 할애합니다. 모하메드는 똑똑한 프로그래머인 당신에게 아이가 책의 문제를 푸는 것을 도와 달라고 부탁합니다.

입력

당신의 프로그램은 하나 또는 그 이상의 테스트 케이스로 검증될 것입니다. 첫 줄에 테스트 케이스의 수를 나타내는 $T$ ($1 \le T \le 100$)가 주어집니다. 다음 $T$개의 줄에는 각각 다섯 개의 정수 $n$, $r1$, $c1$, $r2$, $c2$가 주어집니다. 체스 보드는 $n \times n$ 크기입니다. ($2 \le n \le 10 ^ 9$). ($r1$, $c1$)은 나이트가 시작하는 위치이며 ($r2$, $c2$)는 나이트가 도착해야 하는 위치입니다. ($1 \le r1, c1, r2, c2 \le n$).

출력

각 테스트 케이스에 대해 $n$이 (1부터 시작하는) 테스트 케이스의 번호를 나타내는 "Case n:"을 (큰따옴표 없이) 출력한 후, 나이트가 도착 지점까지 정확히 한 번의 움직임으로 도달할 수 있다면 (큰따옴표를 제외한) "YES" 또는 그렇지 않다면 (큰따옴표를 제외한) "NO"를 출력합니다.

힌트

($r$,$c$)의 나이트는 ($r$은 행 번호, $c$는 열 번호이며, 행은 위에서 아래로 1부터 $n$까지 번호가 매겨지고 열은 왼쪽부터 오른쪽으로 1부터 $n$까지 번호가 매겨집니다.) 체스판 위에서, 8가지 방향 중 한 곳으로 이동할 수 있습니다 (이동 할 곳도 체스판 위라면): ($r-1$,$c+2$), ($r-1$,$c-2$), ($r+1$,$c+2$), ($r+1$,$c-2$), ($r-2$,$c+1$), ($r-2$,$c-1$), ($r+2$,$c+1$), ($r+2$,$c-1$)
