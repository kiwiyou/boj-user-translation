정반대

문제

String 클래스에는 substring이라는 함수가 있다. 여러분은 substring 함수와 정반대의 기능을 하는 프로그램을 작성해야 한다. 문자열에서 주어진 부분 문자열을 뽑아내는 것이 아니라, 문자열에서 주어진 부분 문자열을 제외해야 한다.

입력

첫째 줄에 테스트 케이스의 개수 n이 주어진다. 각 테스트 케이스의 첫째 줄에 문자열과 두 정수 i와 j가 주어진다. (0 ≤ i ≤ j ≤ 문자열의 길이) 제외할 부분 문자열은 i 번째 문자부터 j − 1 번째 문자까지이다. 제외할 부분 문자열의 길이는 j − i이다.

출력

각 테스트 케이스마다 주어진 부분 문자열을 제외한 문자열을 출력한다.
