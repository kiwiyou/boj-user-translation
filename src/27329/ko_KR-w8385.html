문자열 반복

문제

두 개의 동일한 문자열을 연결하여 얻은 문자열을 반복 문자열이라고 합니다.

예를 들어 OIOI, JJJJJJ, JOIOIJOIOI는 반복 문자열이지만, IOOI, JOIIOI는 반복 문자열이 아닙니다.

길이 N 의 문자열 S 가 주어집니다. N 은 짝수이고, S 의 각 문자는 J, O, I 로 이루어져 있습니다.

S 가 반복 문자열이라면 Yes 를, 그렇지 않다면 No 를 출력해야 합니다.

입력

입력은 다음과 같이 주어집니다.

N
S

출력

S 가 반복 문자열이라면 Yes 를, 그렇지 않다면 No 를 출력해야 합니다.

제한

  • 2 ≦ N ≦ 100.
  • N 은 짝수입니다.
  • S 는 길이가 N 입니다.
  • S 의 각 문자는 J, O, I 중 하나입니다.

JOIJOI 는 JOI 를 2 개 연결하여 만들 수 있으므로, 반복 문자열입니다. 따라서 Yes 를 출력합니다.

IOIOIO 는 2 개의 동일한 문자열로 만들 수 없으므로, 반복 문자열이 아닙니다. 따라서 No 를 출력합니다.
