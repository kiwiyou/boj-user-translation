눈사람

문제

추운 겨울, 카시아와 아시아가 공원에 갔습니다. 겨울이라 공원은 눈으로 덮여 있는데, 보다 정확히 말하자면 x L의 눈이 공원 안에 쌓여있습니다. 아시아가 전화를 하는 동안 카시아는 눈사람을 만들기로 했습니다. 아시아가 전화를 끝낼 즈음, 카시아는 이미 k L의 눈으로 공을 하나 만들었습니다.

아시아는 3 개의 눈 공으로 눈사람을 만들고, 추가적으로 공 위에 올려지는 공은 그 아래의 공보다 두 배 작은 크기가 되기를 원합니다. 아시아는 카시아가 화날 수 있기에 카시아가 만든 공을 보다 크거나 작게 만들 수는 없습니다.

여러분, 소녀들은 가능한 가장 큰 눈사람을 만들고 싶습니다. 그들은 그들의 눈사람이 몇 mL의 눈으로 만들어질지 궁금해 합니다.

입력

첫 줄에 각각 공원에 있는 눈의 양과 눈사람을 이루는 하나의 공에 쓰인 눈의 양을 L로 표현한 값인 x와 k (1 ≤ x ≤ 106, 1 ≤ k ≤ 105, k < x)가 주어집니다.

출력

딱 한 줄에 눈사람의 최대 크기를 나타내는 하나의 정수를 출력합니다. (결과를 mL 단위로 출력합시다.)

역주: 눈사람을 만들 수 없을 경우, 0을 출력합니다.
