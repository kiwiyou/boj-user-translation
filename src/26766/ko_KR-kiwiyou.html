하트

문제

프로그래머도 다른 사람들처럼 사랑이 필요하다. 프로그래머 키위새는 사랑하는 고라니에게 고백하기 위해 하트 N 개를 골뱅이(@)로 적어 프로포즈를 하기로 결심했다.

하트 하나는 아래처럼 생겼다.

 @@@   @@@ 
@   @ @   @
@    @    @
@         @
 @       @ 
  @     @  
   @   @   
    @ @    
     @

키위새는 하트가 한 줄에 하나씩 배열된 것을 좋아한다.

N을 입력받아 앞에서 말한 형식으로 하트를 출력하는 프로그램을 만들자.

입력

첫째 줄에 출력된 하트의 개수를 의미하는 양의 정수 N이 입력된다. (1 ≤ N ≤ 100 000)

출력

정확히 N 개의 하트를 위아래로 배열해 출력한다.
