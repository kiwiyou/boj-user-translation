초과 인출

문제

몇몇 은행은 당신이 계좌에 있는 금액보다 더 많은 돈을 인출하려 할 경우 초과 인출 수수료를 청구합니다. 예금액과 인출액이 연속해서 주어질 때 (또 예금액과 인출액이 바로 계좌에 반영된다고 가정할 때), 이 과정에서 초과 인출 수수료를 한 푼도 내지 않으려면 초기 계좌에 최소 얼마(음수 제외)가 있어야 하는지 구합시다.

입력

첫 줄에 거래의 수를 나타내는 하나의 정수 $n$ ($1 \le n \le 1{,}000$)이 입력됩니다.

다음 $n$개의 줄에 하나의 정수 $t$ ($-10^6 \le t \le 10^6$, $t \neq 0$)가 들어옵니다. 이는 거래가 진행된 순서대로 그 값이 주어진 것입니다. 양의 정수는 예금을, 음의 정수는 인출을 뜻합니다. 그 어떤 거래도 동시에 진행되지 않습니다.

출력

초과 인출 수수료를 지불하지 않기 위해 초기 계좌에 있어야 하는 음이 아닌 최소 금액을 나타내는 하나의 정수를 출력합니다.
