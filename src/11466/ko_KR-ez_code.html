알렉스의 종이접기

문제

알렉스는 종이 접기를 좋아합니다. 대부분의 종이 접기는 정사각형 종이로 시작합니다. 알렉스는 어머니께 드릴 선물을 만들려고 합니다. 선물을 만들려면 세 개의 동일한 크기의 정사각형 종이가 필요한데, 알렉스는 하나의 직사각형 종이만 가지고 있습니다. 직사각형 종이에서 정사각형을 잘라낼 수는 있지만, 이 때 잘라낸 종이의 각 변은 직사각형 종이의 각 변에 평행해야 합니다. 알렉스를 도와 잘라내는 정사각형 종이 크기의 최댓값을 구합시다.

입력

두 정수 $h$, $w$가 한 줄에 주어집니다 - 이는 직사각형 종이의 높이와 너비입니다. $(1 \le h, w \le 1000)$

출력

정사각형 종이 한 변의 최대 길이를 나타내는 하나의 실수를 출력합니다. 이 때 이 길이의 종이를 $h \times w$ 크기의 직사각형 종이에서 각 변이 원래 직사각형의 각 변과 평행하도록 세 장 오려낼 수 있어야 합니다.

출력하는 값은 소수점 이후 셋째 자리까지 정확해야 합니다.
