은행

문제

뉴질랜드 키위마을에는 n 개의 은행이 있다. (0 < n < 10000) 각 은행은 원형으로 배치되어 있어 한 은행의 왼쪽과 오른쪽에는 다른 은행이 있고, 0 번 은행부터 시계방향으로 n-1 번 은행까지 존재한다. i 번 은행의 자본금 ki는 -32000 < ki < 32000을 만족한다. 모든 은행의 자본금의 합은 양수임이 알려져 있다. 어떤 은행의 자본금 ki가 음수라면, 은행 요정이 요술을 부려 자본금의 부호를 바꿀 수 있다. 예를 들어 ki = -7이라면 요술을 부리고 나서 ki = 7이 된다. 안타깝지만 이 요술은 이웃한 두 은행에 불행을 가져온다. i 번 은행 양 옆의 은행은 |ki|만큼 자본금이 줄어든다! 예를 들어 왼쪽 은행의 자본금 kL = 5, 오른쪽 은행의 자본금 kR = 11이라면, i 번 은행에 요술을 부린 이후에 kL = -2, kR = 4가 된다.

은행 요정이 모든 은행의 자본금을 0 이상으로 만들기 위해서는 요술을 최소한 몇 번 부려야 할까?

입력

첫째 줄에 은행의 수 n이 주어진다. 둘째 줄에 0 번 은행부터 n-1 번 은행까지, i 번 은행의 자본금 ki가 주어진다. 각 자본금은 공백 하나로 구분되어 있다.

출력

첫째 줄에 문제의 답을 출력한다.
