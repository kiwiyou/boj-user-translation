나머지의 합

문제

2 이상의 정수로 이루어진 중복집합 K과 음이 아닌 정수 n에 대해서 나머지의 합 함수 SK(n)은 다음과 같이 정의된다.

SK(n) = ∑ (k ∈ K ∣ n mod k)

예를 들어 만약 K = {2, 5, 5, 11}라면,

SK(23) = 23 mod 2 + 23 mod 5 + 23 mod 5 + 23 mod 11 = 1 + 3 + 3 + 1 = 8이다.

어떤 K에 대해서도 SK(0) = 0임에 주의하자.

중복집합 K와 1부터 N까지의 n에 대해 SK(n)의 값이 주어진다. 이 문제에서 여러분은 이들을 입력으로 받아 K의 원소의 개수를 출력하고, K의 원소들을 감소하지 않는 순서로 출력해야 한다.

입력

입력은 여러 줄로 주어진다. 첫째 줄에 N이 주어지고, 그다음 줄부터 각 SK(n)의 값이 주어진다. (1 ≤ N ≤ 100; 1 ≤ n ≤ N) 각 줄에는 최대 10 개의 정수가 공백으로 구분되어 주어진다.

출력

첫째 줄에 K의 원소의 개수 m을 출력하고, K의 원소를 감소하지 않는 순서로 출력한다. 같은 값을 가진 원소가 여러 개 있다면 그만큼 출력해야 함에 주의하여라.
