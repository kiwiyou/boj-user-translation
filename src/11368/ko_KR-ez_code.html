중요한 독서

문제

간달프는 절대 반지의 역사를 찾기 위해 기록 보관소 담당 보르헤스가 관리하는 바벨의 도서관에 갔습니다. 도서관에는 - 진실이든 거짓이든, 식견을 넓혀주든 터무니 없든 - 모든 책이 있으며 간달프는 그의 지혜를 이용해 절대 반지에 관한 올바른 설명을 찾아야 합니다.

$C$ 종류의 문자가 있고, 한 줄에 $W$ 개의 문자가 있고, 한 페이지에 $L$ 개의 줄이 있고, 한 책에 $P$ 장의 페이지가 있다고 할 때, 각 책이 정확히 한 권만 있다고 하면 바벨의 도서관에 총 몇 개의 책이 있는지 계산합시다.

예를 들어 도서관에 두 종류의 문자(a, b)가 있고, 한 줄에 3개의 문자가 있고, 한 페이지에 한 개의 줄이 있고, 책에 1개의 페이지가 있는 책만 있다면 도서관에는 아래의 책이 있을 것입니다: aaa aab aba abb baa bab bba bbb

입력

독립된 줄에 각 테스트 케이스가 음이 아닌 정수를 이용해 $C$ $W$ $L$ $P$와 같은 형태로 주어집니다. 처리되지 않는 줄 "0 0 0 0"이 입력의 종료를 알립니다.

출력

각 테스트 케이스에 대해, 도서관의 책의 수를 나타내는 하나의 정수를 새 줄에 출력합니다.
