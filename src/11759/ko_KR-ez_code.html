'병'이 든 기분

문제

피터는 기름을 잔뜩 실은 배를 기다리는데, 문제가 하나 있습니다 (프로그래밍 문제에 나오는 모든 사람처럼 말이죠!). 피터가 가지고 있는 유일한 컨테이너에는 (각각 부피가 같은) 큰 병들과 (역시 각각의 부피는 같지만 용량이) 작은 병들이 있습니다. 배가 싣고 온 기름의 양이 주어질 때, 피터는 아래의 조건을 만족하도록 기름을 병에 담으려 합니다:

  1. 모든 기름을 담는다.
  2. 각 병을 끝까지 채운다.
  3. 병의 개수를 최소로 사용한다.

가지고 있는 병의 크기로는 이 조건을 충족할 수 있지만, 다른 크기의 병을 사용한다면 어떻게 될지 피터는 몇 시간째 고민 중입니다. (확실히 피터는 재밌게 살진 않는 것 같습니다.)

입력

하나의 줄에 세 양의 정수 $s$, $v_1$, $v_2$가 주어지는데, $s \le 10^6$을 만족하는 $s$는 배가 싣고 온 기름의 양이며, $v_1, v_2 \le 10^6$와 $v_1 > v_2$를 만족하는 $v_1$과 $v_2$는 두 종류의 병의 크기를 나타냅니다.

출력

피터의 조건을 만족할 수 있는 $v_1$ 크기의 병의 개수와 $v_2$ 크기의 병의 개수를 출력합니다. 조건을 만족할 수 없다면, Impossible 을 출력합니다.
