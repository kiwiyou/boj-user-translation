문자열 반전

문제

길이 N 의 문자열 S 가 주어집니다.

S 의 A 번째 문자부터 B 번째 문자까지는 배열을 거꾸로 하고 그 외의 문자는 그대로 둔다면 어떤 문자열이 나올까요. 이때 S 의 제일 앞 글자를 첫 번째 글자로, 제일 뒤 글자를 N 번째 문자로 합니다.

문자열 S 와 두 정수 A, B 가 주어질 때, S 의 A 번째 문자부터 B 번째 문자까지 배열을 거꾸로 해 나오는 문자열을 출력하는 프로그램을 작성합시다.

입력

아래의 형식으로 표준 입력이 주어집니다.

N A B
S

출력

S 의 A 번째 문자부터 B 번째 문자까지 배열을 거꾸로 해 나오는 문자열을 한 줄에 출력합니다.
