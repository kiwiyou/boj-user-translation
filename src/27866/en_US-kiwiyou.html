Character

Statement

Given a word $S$ and an integer $i$, print the $i$-th character of $S$.

Input

A string consisting of only alphabet lower and upper case letters, $S$ is given on the first line. The length of $S$ does not exceed $1\,000$.

An integer $i$ is given on the second line. ($1 \le i \le \left|S\right|$)

Output

Print the $i$-th character of $S$.

Note

$\left|S\right|$ denotes the length of $S$.
