시험 점수

문제

서진이는 컴퓨터 과목 시험을 세 번 치렀다. 각 시험 점수는 0 이상 100 이하의 정수이다.

서진이의 성적은 세 시험에서 받은 점수 중 가장 높은 두 점수의 합계로 계산된다.

세 시험에서 받은 점수 A, B, C가 주어질 때, 서진이의 성적을 출력하는 프로그램을 작성하여라.

입력

입력은 아래와 같이 주어진다.

A B C

출력

세 시험에서 받은 점수 중 가장 높은 두 점수의 합계를 출력한다.

제한

  • 0 ≤ A ≤ 100
  • 0 ≤ B ≤ 100
  • 0 ≤ C ≤ 100
  • 入力はすべて整数である．
  • 점수는 높은 것부터 90, 80, 70이므로, 구하는 합은 90 + 80 = 170이 된다.
  • 점수는 높은 것부터 100, 70, 70이므로, 구하는 합은 100 + 70 = 170이 된다.
