Fenced In (Platinum)

문제

농부 존은 자신이 키우는 소들 중 많은 수가 광장공포증이 있다는 것을 알게 되었다. (광장공포증이란 광장과 같은 넓게 트인 공간을 두려워하는 질환이다.) 그는 소들이 편안하게 풀을 뜯어먹을 수 있도록, 수직 울타리들과 수평 울타리들을 세워서 자신의 넓은 들판을 여러 개의 작은 공간으로 나눴다.

넓은 들판은 $(0,0)$과 $(A,B)$를 꼭짓점으로 하는 직사각형 모양이다. FJ builds \(n\) vertical fences (\(\)) at distinct locations \(a_1 \ldots a_n\) (\(0 < a_i < A\)); each fence runs from \((a_i, 0)\) to \((a_i, B)\). 농부 존은 $n$($0 \le n \le 25\,000$)개의 수직 울타리들을 서로 다른 위치 $a_1 \ldots a_n$에 세웠다. 각 수직 울타리들은 $(a_i,0)$부터 $(a_i,B)$까지 이어진다. He also builds \(m\) horizontal fences (\(0 \leq m \leq 25,000\)) at locations \(b_1 \ldots b_m\) (\(0 < b_i < B\)); each such fence runs from \((0, b_i)\) to \((A, b_i)\). Each vertical fence crosses through each horizontal fence, subdividing the large field into a total of \((n+1)(m+1)\) regions.

Unfortunately, FJ completely forgot to build gates into his fences, making it impossible for cows to leave their enclosing region and travel around the entire field! He wants to remedy this situation by removing pieces of some of his fences to allow cows to travel between adjacent regions. He wants to select certain pairs of adjacent regions and remove the entire length of fence separating them; afterwards, he wants cows to be able to wander through these openings so they can travel anywhere in his larger field.

For example, FJ might take a fence pattern looking like this:

+---+--+
|   |  |
+---+--+
|   |  |  
|   |  |
+---+--+

and open it up like so:

+---+--+
|      |  
+---+  +  
|      |  
|      |
+---+--+

Please help FJ determine the minimum total length of fencing he must remove to accomplish his goal.

입력

첫 번째 줄에는 $A$, $B$, $n$, $m$이 차례대로 공백으로 구분되어 주어진다. ($1 \le A, B \le 1\,000\,000\,000$)

두 번재 줄부터 $n$개의 줄에 걸쳐 $a_1, \ldots ,a_n$이 하나씩 차례대로 주어진다.

이어서 $m$개의 줄에 걸쳐 $b_1,\ldots,b_m$이 하나씩 차례대로 주어진다.

출력

농부 존이 제거해야 하는 울타리의 길이의 최솟값을 출력하라. 답이 32-bit 정수형보다 커질 수 있으므로 64-bit 정수형을 사용해야 함에 유의하라.
