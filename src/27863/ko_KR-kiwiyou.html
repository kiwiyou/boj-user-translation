유랑 계획

문제

유랑길은 하루 조금 안 되는 동안 움직일 수 있는 거리만큼 떨어진 야영지들로 구성된다. 야영지에는 물이 거의 없을 수도 있고, 직접 마시거나 동물에게 먹일 물이 풍부한 오아시스가 있을 수도 있다. 유랑은 항상 오아시스에서 출발해서 오아시스에서 끝나며, 한 번 지난 야영지로는 다시 돌아가지 않는다. 유랑은 도착점 오아시스와 거기까지 도착하기 위해 걸리는 시간으로 정의된다. 예를 들어, 오아시스가 야영지 2, 3, 5, 7, 11 번 등에 있고, 일행이 7 번 야영지에서 10 일 안에 다른 일행을 만나고 싶다면, 일행은 3 일을 기다렸다 7 번 야영지로 바로 가거나, 지금 바로 떠나 7 번 야영지에서 3 일을 기다리거나, 2, 3, 5 번 야영지에서 각각 1 일간 기다릴 수 있다. 유랑 계획은 각 밤에 쉬는 야영지의 번호로 이루어진 수열이다. 예를 들어 2, 3, 5 번 야영지에서 각각 1 일간 쉬는 계획은 1, 2, 2, 3, 3, 4, 5, 5, 6, 7 로 나타낼 수 있다.

유랑길에 있는 오아시스의 위치와 유랑의 도착 지점, 걸리는 시간이 주어졌을 때 가능한 서로 다른 유랑 계획의 가짓수를 출력하여라.

예를 들어 오아시스가 2, 3, 5, 7, 11 번에 있다면, 7 일 동안 5 번 야영지까지 가는 경우는 다음 10 가지가 있다. (0은 출발 지점에서 쉼을 나타낸다.)


0012345, 0122345, 0123345, 0123455, 1222345, 1223345, 1223455, 1233345, 1233455, 1234555

입력

입력은 여러 줄로 주어진다. 첫째 줄에는 오아시스의 개수 N과 유랑 경로를 찾을 횟수 M이 주어진다. (5 ≤ N ≤ 20; 1 ≤ M ≤ 10)

둘째 줄에는 각 N 개의 오아시스까지 가는 데 걸리는 날의 수 On이 증가하는 순으로 (On-1 ≤ On) 주어진다. (1 ≤ On ≤ 60)

나머지 M 개의 줄에는 도착 지점 오아시스의 번호 Dm과 거기까지 가는 데 걸리는 시간 Tm이 주어진다. (1 ≤ Dm ≤ N; O[Dm] ≤ Tm ≤ 60)

출력

총 M 개의 줄을 출력한다. k 번째 줄에는 Dk 번 오아시스까지 Tk 일 안에 도착하는 서로 다른 유랑 계획의 수를 출력한다.
