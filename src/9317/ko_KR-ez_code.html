모니터 DPI

문제

화면 해상도는 꽤 빠르게 좋아지고 있고 합리적인 소비자인 당신은 당신이 사려고 하는 모니터의 DPI(인치 당 도트 수)를 비교하려 합니다. 안타깝게도 제조사는 DPI를 표기하지 않고 대각선 길이와 해상도만 각각 인치 단위와 너비×높이 단위로 제공합니다. 모든 모니터의 너비 대 높이 비율은 16:9로 가정합시다.

다음과 같이 가정합시다.

\[W = \text{모니터의 너비 (인치)}\] \[H = \text{모니터의 높이 (인치)}\] \[D = \text{모니터의 대각선 길이(인치)}\]

$D$는 다음과 같으며

\[D = \sqrt{W^2+H^2}\]

그리고

\[H = \frac{9}{16}W\]

따라서

\[D = \sqrt{W^2 + \left(\frac{9}{16}W\right)^2} = \sqrt{\left(1+\frac{9^2}{16^2}\right)W^2} = \sqrt{1+\frac{9^2}{16^2}}W \]\[ W = \frac{16 \times D}{\sqrt{337}}  \]\[ \text{DPI}_{\text{너비}} = \frac{\text{해상도}_{\text{너비}}}{W} \]\[ \text{DPI}_{\text{높이}} = \frac{\text{해상도}_{\text{높이}}}{H} \]

입력

각 입력에 세 수로 십진수 $D$, 정수 \(\text{해상도}_{\text{너비}}\)와 정수 \(\text{해상도}_{\text{높이}}\)가 주어집니다. 0 세 개가 입력의 종료를 나타냅니다.

출력

각 테스트 케이스(한 줄의 입력)에 대해 아래와 같이 소수 두 자리까지 반올림한 가로 DPI와 세로 DPI 값을 계산합니다:

Horizontal DPI: XXX.XX

Vertical DPI: XXX.XX
