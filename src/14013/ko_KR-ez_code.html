단위 변환

문제

"'2kg'을 영어로 어떻게 말해?" - 두두, 2015.

두두는 항상 측량 및 단위 변환을 어려워 했습니다. 그를 도와줄 수 있습니까?

입력

입력은 두 숫자 $x$와 $y$가 포함된 한 줄로 시작되는데, 이는 "$x$개의 단위 $A$"가 "$y$개의 단위 $B$"와 동일함을 의미합니다. 예를 들어, $A$가 "킬로그램"이고 $B$가 "파운드"라면 가능한 하나의 경우로는 $x = 3.25$, $y = 7.165024$가 있습니다.

다음 줄에 변환해야 하는 데이터의 수를 나타내는 하나의 정수 $N$이 주어집니다.

각 $N$개의 줄은 "$z$ $q$"의 형태로 주어지며 $z$는 숫자, $q$는 '$A$' 또는 '$B$' 중 하나입니다.

  • $1 \le N \le 100\,000$
  • $x$, $y$ 는 양수입니다.
  • $z$는 음이 아닌 수입니다.

출력

$N$개의 줄에 숫자를 각각 출력합니다. 구체적인 내용은 예시 입출력을 확인합시다.

힌트

$10^{-4}$보다 작은 실수오차는 정답으로 인정됩니다.

  • $3.25 A = 7.165024 B$
  • $1 A = 2.20462277 B$
  • $0 B = 0 A$
  • $2.1 B = 0.952543913321 A$
  • $0 A = 0 B$
