사라의 장난감

문제

5살 사라는 봉제 인형을 많이 가지고 있습니다. 인형은 원래라면 침실에 있어야 하지만, 각 밤에 침실로 돌아오지 못하는 인형도 있습니다.

잠에 들 때, 사라는 방에 있는 봉제 인형을 여분 침대에 놓습니다. 사라는 봉제 인형에게 밤을 같이 보낼 친구를 만들어주는 것을 좋아해 인형에 짝을 맺어줍니다. 가끔 인형이 짝수가 아니어서 세 개 한 묶음을 만들어야 할 때도 있습니다.

당신은 특정 밤에 사라가 장난감을 어떻게 정리했는지 알려주는 프로그램을 작성해야 합니다.

입력

입력은 여러 줄로 주어지는데, 각 줄은 사라의 집의 한 밤을 나타냅니다. 각 줄에는 두 개의 정수가 하나의 공백으로 구분되어 주어집니다. 첫 번째 수는 그녀가 그 시점에 가지고 있는 봉제 인형의 수를 나타냅니다. 두 번째 수는 집 어딘가에 있지만 밤에 사라의 침실로 돌아오지 못한 봉제 인형의 수입니다.

마지막 줄로 0 0이 주어집니다. 이 줄은 처리하지 않습니다.

출력

각 밤에 대해 하나의 공백을 두고 두 수를 출력합니다. 첫 번째 수는 쌍의 수를 나타내고 두 번째 수는 0 또는 1입니다. 0은 세 개로 된 묶음이 없음을 의미하고, 1은 세 개로 된 묶음이 하나 있음을 의미합니다.
