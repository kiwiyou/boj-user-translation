L번째 K번째 숫자

문제

가로 방향으로 일렬로 배열된  N 장의 카드가 있다. 왼쪽으로부터  i 번째(1 ≤ i ≤ N)의 카드에는 정수  ai 가 적혀있다.

JOI 군은 이 카드를 이용하여 다음과 같은 게임을 한다. 연속한  K 장 이상의 카드열을 선택하고, 다음 작업을 수행한다.

  • 선택한 카드열을 카드에 적힌 정수가 작은 순으로 왼쪽부터 늘어놓는다.
  • 늘어놓은 카드 중 왼쪽으로부터  K 번째 카드에 적힌 정수를 종이에 쓴다.
  • 선택한 카드를 모두 원래 위치에 되돌려 놓는다.

이 작업은 연속하는  K 장 이상의 모든 카드열에 대해 수행된다. 즉, 1 ≤ l ≤ r ≤ N 과  K ≤ r - l + 1 을 만족하는 모든  (l,r) 에 대해 al, al+1, ..., ar  중  K 번째로 작은 정수를 종이에 쓴다는 것이다.

이렇게 종이에 적은 정수를 왼쪽으로부터 작은 순서로 정렬한다. 적힌 정수 중 왼쪽에서  L 번째 것이 이 게임에서 JOI 군이 얻는 점수이다. JOI 군의 점수를 구하라.

입력

입력은 아래와 같은 형식으로 표준 입력으로 주어진다.

N K L
a1 a2 ... aN

출력

JOI 군의 점수를  1 줄로 출력하라.

제한

  • 1 ≤ N ≤ 200000
  • 1 ≤ K ≤ N
  • 1 ≤ ai ≤ N
  • 1 ≤ L
  • JOI 군이 종이에 적는 정수는  L 개 이상이다.

서브태스크

번호배점제한
16

N ≤ 100

233

N ≤ 4000

361

추가 제한이 없다.

1 ≤ l ≤ r ≤ N (= 4) 와  K (= 3) ≤ r - l + 1 을 만족하는  (l,r) 은 (1,3), (1,4), (2,4) 의  3 가지이다.

이  (l,r) 에 대해, al, al+1, ..., ar  중  3 번째고 작은 정수는 각각 4, 3, 3 이다.

이 중  L (= 2) 번째로 작은 정수는  3 로, JOI 군의 점수는  3 이다. 같은 정수가 여러 개 있을 때에도 중복하여 세야 한다는 것에 주의하라.

JOI 군이 적은 정수는

  • (l,r) = (1,3) 에 대해 5
  • (l,r) = (1,4) 에 대해 2
  • (l,r) = (1,5) 에 대해 2
  • (l,r) = (2,4) 에 대해 5
  • (l,r) = (2,5) 에 대해 4
  • (l,r) = (3,5) 에 대해 4

이다. 이 중  L (= 3) 번째로 작은 정수는  4 이다.
