수열의 길이

문제

음이 아닌 십진수 정수를 $l$부터 $r$까지 차례로 적은 문자열 $a$가 있다고 하자. 예를 들면 $l=3$, $r=10$인 경우에는 $a=345678910$가 된다.

이제 이런 방식으로 $[l,r]$ $(0 \le l \le r \le 10^{18})$을 골라서 구성한 $a$의 길이가 정확히 $S$가 되면서 $[l,r]$에 속하는 수가 최대한 많도록 하고자 한다. 이러한 $[l,r]$을 찾아보자.

입력

첫째 줄에 정수 $S$가 주어진다. ($1 \le S \le 10^{18}$).

출력

첫째 줄에 가능한 제일 긴 $[l,r]$에 속하는 수의 개수를 출력한다. 만약 그러한 구간이 없다면 $-1$을 출력한다.

가능한 구간이 존재한다면 둘째 줄에 $l$, $r$을 출력한다.

최적의 값이 여럿이라면 아무거나 출력한다.
