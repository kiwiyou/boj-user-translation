N일 간의 크리스마스 선물

문제

수정이는 매년 크리스마스 선물을 받는다! 그런데 수정이가 크리스마스 선물을 받는 형식은 조금 독특하다.

  • $1$ 번째 크리스마스에 수정이는 $1$ 개의 선물을 받는다.
  • $2$ 번째 크리스마스에 수정이는 $(1 + 2)$ 개의 선물을 받는다.
  • $3$ 번째 크리스마스에 수정이는 $(1 + 2 + 3)$ 개의 선물을 받는다.
    • ...
  • $N$ 번째 크리스마스에 수정이는 $(1 + 2 + 3 + ... + N)$ 개의 선물을 받는다.

문득 수정이는 $1$ 번째 크리스마스부터 $N$ 번째 크리스마스까지 받은 선물의 개수가 궁금해졌다. 하지만 수정이는 선물 더미 속에 파묻혀버렸기 때문에, 선물의 개수를 셀 겨를이 없다. 여러분이 대신 수정이가 받은 선물의 개수를 계산해 주자!

입력

입력은 여러 개의 테스트 케이스로 이루어져 있다.

각 테스트 케이스는 하나의 줄로 이루어져 있으며, 각 줄에는 정수 $N$ $(1 \le N \le 1\;000\;000)$ 이 입력으로 주어진다.

입력의 마지막 줄에는 $0$ 이 입력으로 주어지며, $0$ 이 입력으로 들어왔을 경우, 여러분의 프로그램은 더 이상의 출력을 하지 않고 종료되어야 한다.

출력

각 테스트 케이스마다 수정이가 $1$번째 크리스마스부터 $N$번째 크리스마스까지 받은 선물의 개수를 한 줄에 하나씩 출력한다.
