깃발

문제

COCI를 준비하느라 지친 비토는 $20$ 분 간격으로 총 $3$ 시간밖에 자지 못한데다 패트릭과 요십의 장난이 이어지는 통에 마침내 잠에 들었다.

믿음직한(?) 친구들이 말을 듣지 않으니 언제나 평화주의를 외쳤던 비토는 포기한다는 의미로 $n$ 개의 백기가 나오는 꿈을 꿨다. 백기는 한 변이 땅과 평행한 직각삼각형 모양으로 펄럭거리고 있었다. 깃발이 너무 많아 아침에 비토는 각 $i$번째 깃발의 빗변의 길이가 $r_i$였고, 깃발의 높이를 합하면 많아야 $S$였다는 것만을 기억할 수 있었다.

잠도 깬 김에 비토는 해변에서 투쟁하며 절대 항복하지 않기로 결심했다! 먼저 가까운 페인트 가게로 달려가 다음에 $n$ 개의 깃발이 나오는 꿈을 꾸면 모두 색칠해 버리기로 했다! 하지만 이내 페인트를 얼마나 사야 할지 고민에 빠졌다. 그래서 비토는 여러분이 $n$ 개의 깃발이 차지하는 총 면적의 최댓값을 계산해 주길 바란다.

입력

첫째 줄에 깃발의 수 $n$과 깃발의 높이의 합의 최댓값 $S$가 주어진다. $(1 \le n \le 100\,000;$ $1\le S\le 10^10)$

둘째 줄에 $n$개의 정수 $r_i$가 주어진다. $(1 \le r_i \le 100\,000)$

출력

첫째 줄에 깃발이 차지하는 전체 넓이의 최댓값을 출력한다. 절대/상대 오차는 $10^{-6}$까지 허용한다.

서브태스크

번호배점제한
141

$n ≤ 100$

222

$n ≤ 1000$

347

추가적인 제약 조건이 없다.

힌트

두 번째 예제에서, 깃발이 차지하는 넓이는 직각삼각형의 각 변의 길이가 $6$, $8$, $10$일 때 $24$로 최대이다.
