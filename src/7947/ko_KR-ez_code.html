콘서트

문제

유명한 연예인의 콘서트 팀이 난관을 맞이했습니다. 연예인은 콘서트 도중 고유한 특정 색의 조명을 받고 싶어합니다. 콘서트 기술 팀은 연예인을 비추기 위해 10개의 스포트라이트를 사용하는데, 다양한 방향에서 엄선된 색의 조명을 동시에 비춰 무대 전체에는 색이 합성되어 모두 비춰지도록 합니다. 당신은 기술 팀을 도와 연예인이 있는 무대 중심에 어떤 색이 비춰질지 계산해야 합니다.

입력

첫 줄에 데이터의 수를 나타내는 정수 $z$가 주어집니다. 하나의 데이터는 아래와 같이 주어집니다:

10개의 줄 각각에 세 정수 $r$, $g$, $b$가 공백으로 주어져 ($0 \le r, g, b \le 255$) 해당하는 스포트라이트의 삼원색 배합을 나타냅니다. 연예인을 비추는 빛의 색은 사용된 10개 스포트라이트의 색의 단순 평균이라고 가정합니다.

출력

각 데이터에 대해, 스포트라이트의 빛의 평균인 연예인에게 비춰지는 빛의 색의 삼원색 구성을 나타내는 $r$, $g$, $b$를 출력합니다. 결과 값은 가장 가까운 정수로 반올림한, 즉 소수 첫째 자리 1부터 4까지는 내림하고 5부터 9까지는 올림한 값입니다.
