선물

문제

싱가포르 올림픽 수영 선수인 조셀린 여의 코치는 그녀의 접영 성적이 염려됩니다. 코치는 매일 그녀의 기록을 밀리초 (밀리초는 1초의 1 / 1000입니다.) 단위로 기록한 후, 기록이 전 날보다 일정 밀리초 이상 좋으면 작은 격려 선물을 주기로 했습니다.

매일 측정한 기록이 주어질 때, 조셀린이 몇 개의 선물을 받을 수 있는지 계산합시다.

입력

첫 줄에 두 정수 $n$과 $k$가 주어지는데, $n(3 \le n \le 100)$은 날의 수, $k(0 < k \le 100,000)$는 (밀리초 단위로 주어진) 개선 목표입니다. 조셀린이 전날에 비해 $k$밀리초 이상 기록을 단축할 때마다, 그녀는 코치에게 선물을 받습니다. 다음 $n$개의 줄에 조셀린의 그날 기록을 밀리초 단위로 나타낸 하나의 정수 $t(0 < t \le 100,000)$가 주어집니다. $n$개의 기록은 시간 순으로 주어집니다.

출력

조셀린이 받은 선물의 수를 나타내는 하나의 정수를 출력합니다.
