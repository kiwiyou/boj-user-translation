기다리는 시간

문제

당신은 친구들과 문자 메시지를 주고 받습니다. 메시지를 너무 많이 받아, 친구가 당신의 답장을 얼마나 오래 기다려야 하는지 계산하려 합니다.

당신의 휴대폰은 아래의 두 가지 입력을 사용해 순서대로 메시지를 받거나 보냅니다:

  • R X 는 X 번 친구에게 메시지를 받았음을 의미합니다.
  • S X 는 X 번 친구에게 메시지를 보냈음을 의미합니다.

당신의 휴대폰은 메시지를 받거나 보낼 때 별도의 시간이 걸리지 않으며, 위에 제시된 연속된 각 입력 쌍에 대해 아래의 입력이 주어집니다:

  • 하나의 입력 W X 는 입력 쌍 사이에 주어져 그 입력이 X 초 간격을 두고 진행됐음을 의미하며,
  • 입력이 없을 경우 1초 간격을 두고 입력이 진행됐음을 의미합니다.

몇 가지 문자 메시지 매너는 아래와 같습니다:

  • 당신은 받은 메시지에 대한 답장만 보냅니다;
  • 당신은 임의의 친구의 임의의 메시지에 대해 최대 한 개의 답장만 보냅니다;
  • 당신의 친구는 당신이 이전 메시지에 답장하기 전까지 연속해 메시지를 보내지 않습니다.

메시지를 기다리는 시간은 당신이 메시지를 받은 시간과 답장을 하는 시간 사이 흐른 시간 간격입니다. 만약 X 번 친구가 자신이 보낸 각 메시지에 대해 답장을 받았다면, X 번 친구가 기다린 총 시간은 X 번 친구의 모든 메시지의 기다린 시간의 합입니다. 그 외의 경우, X번 친구의 기다린 총 시간은 −1입니다.

당신은 각 친구에 대해 총 기다린 시간을 계산해야 합니다.

입력

입력은 하나의 정수 M 이 주어진 후 M 개의 줄이 주어지는데 (1 ≤ M ≤ 20), 각 줄은 (W, R, or S) 하나의 문자와 이후 하나의 공백, 하나의 정수 X 의 (1 ≤ X ≤ 100) 형태입니다. M개의 줄은 위에 제시된 입력을 (순서대로) 나타냅니다.

출력

한 줄에 각 친구가 메시지를 기다린 시간을 X 가 친구의 번호, T 가 X 번 친구가 메시지를 기다린 총 시간을 나타내는 X T 형식으로 출력합니다. 친구 번호 기준 오름차순으로 줄을 출력합니다.
