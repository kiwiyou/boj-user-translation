못

문제

JOI 군은 판자에 못을 박으면서 놀고 있다. 아래의 그림과 같이, JOI 군은 한 변에 N개의 못이 놓인 정삼각형 모양으로 못을 박아놓았다. 위에서부터 a행 (1 ≤ a ≤ N) 에는 a개의 못이 나란히 박혀있다. 그 중 왼쪽에서 b번째 (1 ≤ b ≤ a) 의 못을 (a, b) 라 한다.

그림 1: 못을 박아놓은 방법(N = 5 의 경우)

못을 꼭짓점으로 하는 삼각형에 대해 '각 변이 전체 정삼각형의 변과 평행하고, 전체 정삼각형과 같은 방향'일 때, 이런 정삼각형을 '좋은 정삼각형'이라고 한다. 즉, '좋은 정삼각형'이란 3개의 못 (a, b), (a + x, b), (a + x, b + x) 를 꼭짓점으로 하는 정삼각형을 의미한다.（이때 a, b, x 는 1 ≤ a < N, 1 ≤ b ≤ a, 1 ≤ x ≤ N − a 를 만족한다).

JOI 군은 고무줄을 사용하여 '좋은 정삼각형'을 둘러싸기로 했다.

그림 2: 고무줄로 '좋은 정삼각형'을 둘러싼 방법의 예시

정삼각형의 한 변에 놓인 못의 개수 N, JOI 군이 가지고 있는 고무줄의 개수 M, 그리고 M개의 고무줄로 '좋은 정삼각형'을 둘러싸는 방법이 주어질 때, 1개 이상의 고무줄로 둘러싸인 못의 개수를 구하는 프로그램을 작성하라.

입력

표준 입력으로 다음 데이터를 입력받는다.

  • 첫째 줄에는 정수 N, M이 공백을 사이에 두고 주어진다. N은 정삼각형의 한 변에 놓인 못의 개수를, M은 JOI 군이 갖고 있는 고무줄의 개수를 각각 나타낸다.
  • 이어지는 M줄에는 고무줄로 '좋은 정삼각형'을 둘러싸는 방법이 주어진다. i + 1 번째 줄 (1 ≤ i ≤ M) 에는 정수 Ai, Bi, Xi (1 ≤ Ai < N, 1 ≤ Bi ≤ Ai, 1 ≤ Xi ≤ N − Ai) 가 공백을 사이에 두고 주어진다. 이는 i번째 고무줄이 3개의 못 (Ai, Bi), (Ai + Xi, Bi), (Ai + Xi, Bi + Xi) 을 꼭짓점으로 하는 '좋은 정삼각형'을 둘러싸고 있음을 의미한다.

출력

표준 출력으로 1개 이상의 고무줄로 둘러싸인 못의 개수를 한 줄에 출력하라.

제한

  • 2 ≤ N ≤ 5000 한 변에 놓인 못의 개수
  • 1 ≤ M ≤ 500000 (= 5 × 105) 고무줄의 개수

힌트

이 예제는 그림 2와 같이 '좋은 정삼각형'을 둘러싼 방법에 해당한다. 이 예제에서 (1, 1), (4, 4), (5, 5)를 제외한 12개의 못이 1개 이상의 고무줄로 둘러싸여 있다.
