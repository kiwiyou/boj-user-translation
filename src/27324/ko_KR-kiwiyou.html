문제

십진법으로 두 자리인 정수 N이 주어진다.

N의 십의 자리 숫자와 일의 자리 숫자가 같은 경우 1을, 아니면 0을 출력하자.

입력

입력은 아래 형식으로 주어진다.

 N

출력

N이 주어진다.

N의 십의 자리 숫자와 일의 자리 숫자가 같은 경우 1을, 아니면 0을 출력한다.

제한

  • 10 ≦ N ≦ 99
  • N은 정수이다.

22는 십의 자리 숫자가 2이고, 일의 자리 숫자가 2로 둘이 같으므로 1을 출력한다.

10는 십의 자리 숫자가 1이고, 일의 자리 숫자가 0으로 둘이 다르므로 0을 출력한다.
