나단

문제

팰린드롬 이름을 가진 나단은 젊은 소프트웨어 개발자의 프로젝트를 지원하는 성공한 사업자입니다. 올해 그는 (크로아티아 화폐 단위인) $K$ 쿠나를 각 프로젝트에 최소 1쿠나씩 돌아가고 모든 프로젝트가 서로 다른 쿠나를 받도록 $N$개의 프로젝트에 배분하려 합니다. 배분은 언제나 가능합니다.

$N$과 $K$가 주어질 때, $K$ 쿠나를 $N$개의 프로젝트에 배분하는 가능한 방법 하나를 출력하는 프로그램을 작성합시다.

입력

첫 줄에 문제에 주어진 양의 정수 $K$ $(100 \le K \le 1\,000\,000)$가 주어집니다. 둘째 줄에 문제에 주어진 양의 정수 $N$ $(1 \le N \le 100)$이 주어집니다.

출력

분배 방식대로 돈을 나눌 경우 첫 번째 프로젝트가 받을 돈을 첫 번째 줄에, 두 번째 프로젝트가 받을 돈을 두 번째 줄에, 같은 방식으로 $N$번째 프로젝트가 받을 돈을 $N$번째 줄에 출력합니다.
