수박

문제

슈퍼 히어로라면 큰 수박 4개를 들고 다닐 줄 알아야 합니다. 슈퍼 히어로는 양쪽의 무게 차이가 최소가 되도록 수박을 양손에 나누려고 합니다.

양손 무게 차이의 최솟값을 구하는 프로그램을 작성합시다.

입력

첫 줄에 하나의 공백을 기준으로 네 양의 정수가 입력됩니다. - 각 수박의 무게를 kg으로 나타낸 값입니다.

출력

첫 줄에 슈퍼 히어로의 왼손과 오른손에 들려 있는 수박의 무게의 차의 최솟값을 나타내는 음이 아닌 정수 하나를 출력합니다.

제한

  • $0 \le \text{수박 무게} < 100$

힌트

예 1의 설명: 수박은 왼손에 4+7kg, 오른손에 2+8kg이 되도록 분배됩니다. 각 손에 들린 수박의 무게 합은 11과 10입니다. 차이는 1입니다.

예 2의 설명: 왼손은 각각 2kg인 수박 3개를, 오른손은 8kg인 수박을 듭니다.
