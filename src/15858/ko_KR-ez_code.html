간단한 연산

문제

세 정수 a, b, c가 주어집니다. (1 ≤ a, b, c ≤ 109) 오차가 10-6 이하가 되도록 a ⋅ b ∕ c를 계산합시다.

입력

유일한 줄에 세 정수 a, b, c가 하나의 공백을 두고 주어집니다.

출력

하나의 소수를 출력합니다. 이 값은 a ⋅ b ∕ c와 최대 10-6 의 값 차이를 지닐 수 있습니다, 즉 |x − a ⋅ b ∕ c| ≤ 10-6 입니다.

서브태스크 1 (25점)

  • 1 ≤ a, b, c ≤ 10

서브태스크 2 (25점)

  • 1 ≤ a, b, c ≤ 10$\,$000

서브태스크 3 (25점)

  • 1 ≤ a, b ≤ 109, c=1

서브태스크 4 (25점)

  • 1 ≤ a, b, c ≤ 109
