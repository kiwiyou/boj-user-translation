유산

문제

부동산 부자가 이제 너무 늙어 실없는 얘기를 하는 삶의 단계에 이르렀습니다. 자연스럽게, 그녀가 사랑하는 $N$명의 딸이 어머니의 유산을 두고 논의를 시작했습니다.

막내는 논의만 하는 것에 지쳐, 편하게 유산의 일부를 낚아 채기로 결정했습니다. 막내는 어머니가 금 펜던트를 어디에 숨겨 놓았는지 정확히 압니다 - 복도 거울 옆 세 번째 서랍 속 두꺼운 양말 안입니다! 약삭빠른 딸은 펜던트 묶음을 찾아 동일한 크기의 $N$개 부분으로 나누고, 그녀 몫을 가져간 후 나머지를 양말에 넣었습니다. 펜던트 묶음을 동일하게 $N$개 묶음으로 나눌 수 없다면, 묶음을 거의 동일하게 나눕니다: 각 묶음은 다른 묶음에 비해 최대 1개 더 많습니다. 이 경우, 딸은 자신의 몫으로 보다 적은 크기의 묶음 중 하나를 가져갑니다.

나머지 딸은 이 (악한) 행동을 알고 남은 펜던트의 수를 센 다음, 막내가 그녀 몫을 가져가기 전 양말 안에 몇 개의 펜던트가 있었는지 알아내려 합니다. 이 문제에 답하는 것은 당신의 몫입니다. 가능한 답이 하나 이상이라면, 그 중 최솟값과 최댓값을 출력합니다.

입력

첫째 줄에 딸의 수를 나타내는 정수 $N$ ($2 \le N \le 15$)이 입력됩니다.

둘째 줄에 남은 펜던트의 수인 정수 $O$ ($N \le O \le 100$)가 입력됩니다.

출력

유일한 줄에 두 정수를 출력합니다: 이는 총 펜던트 개수의 최솟값과 최댓값입니다.

힌트

예제 1에 대한 설명: 두 딸이 있으므로, 막내 딸이 메달리온의 절반을 가져갔다는 뜻입니다. 처음에 9개의 메달이 있었다면 막내가 4개를 가져갔으므로 5개가 남습니다. 만일 처음에 10개의 메달이 있었다면 막내는 5개를 가져갔고 5개가 남습니다. 따라서 가능한 답은 9와 10입니다.
