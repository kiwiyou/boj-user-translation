객관식 시험

문제

선생님은 객관식 시험을 좋아합니다. 한 가지 이점은 채점하기가 너무 편하다는 것입니다! 다른 이점은 학생들에게 A, B, C, D, E라는 선택지가 있으므로, 자신이 맞출 확률이 못해도 20%라고 믿는 것입니다.

선생님이 객관식 시험을 채점하는 데 사용할 수 있는 프로그램을 작성하세요.

입력

첫째 줄에 정수 $N(0 < N < 10000)$이 주어지고, 추가로 $2N$개의 입력 라인이 주어집니다. $2N$개의 입력은 $N$개의 학생이 제출한 답(A, B, C, D, E 중 하나)과 $N$개의 정답으로 구성됩니다. 정답이 주어지는 순서는 학생이 제출한 답의 순서와 동일합니다(즉, $i$번째 줄이 학생이 제출한 답일 경우, $N + i$번째 줄에는 해당 문제의 정답이 있습니다).

출력

학생이 정답을 맞힌 문제의 수에 해당하는 정수 $C(0 \le C \le N)$을 출력합니다.
