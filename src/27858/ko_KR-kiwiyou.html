피타고라스 쌍 세기

문제

피타고라스 쌍은 다음 조건을 만족하는 세 양수의 집합이다.

a² + b² = c²

기약 피타고라스 쌍은 a, b, c의 최대공약수가 1인 피타고라스 쌍을 말한다.

양수 n을 받아 다음 네 가지 항목의 개수를 세는 프로그램을 작성하여라.

  1. n을 빗변의 길이 ( c ) 로 하는 서로 다른 기약 피타고라스 쌍의 개수
  2. n을 빗변의 길이 ( c ) 로 하는 기약이 아닌 피타고라스 쌍의 개수
  3. n이 밑변 또는 높이 ( a, b ) 인 기약 피타고라스 쌍의 개수
  4. n이 밑변 또는 높이 ( a, b ) 인 기약이 아닌 피타고라스 쌍의 개수

같은 a, b, c에 대해서 b, a, c는 a, b, c와 같은 경우로 센다.

예를 들어 n = 65일 때 위 조건을 만족하는 경우는 다음과 같다.

  1. 33, 56, 65; 63, 16, 65
  2. 39, 52, 65; 25, 60, 65
  3. 65, 72, 97; 65 2112 2113
  4. 65, 420, 425; 65, 156, 169

입력

첫째 줄에 양수 n이 주어진다. (3 ≤ n ≤ 2500)

출력

한 줄에 위에서 말한 네 가지 항목의 개수를 공백으로 구분하여 출력한다.
