개미 가두기

문제

당신의 친구 오스카는 최근 개미 농장을 샀습니다. 그런데 실수로 개미 농장을 바닥에 쏟아 개미가 바닥 이곳저곳을 돌아다니고 있습니다! 당신은 하나의 직사각형으로 둘러싸 개미가 미친 듯이 돌아다니지 않게 하려 합니다. 하지만 기술적 문제로 인해 당신의 울타리는 X축과 Y축에 평행해야 합니다. 아래 그림은 개미를 가두는 울타리의 예시입니다 (테두리의 두께는 단지 시각적인 목적입니다.)

모든 개미를 가둘 수 있는 울타리의 최소 둘레와 최소 면적은 얼마일까요?

입력

첫 줄에 테스트 케이스의 수가 입력됩니다. 각 테스트 케이스는 개미의 수를 나타내는 $N$ $(1 \le N \le 100)$이 입력되며 시작됩니다. 다음 $N$개의 줄에 개미의 위치를 나타내는 두 소수 $X$, $Y$ $(-1\,000.0 \le X, Y \le 1\,000.0)$가 입력됩니다. 개미는 넓이가 없는 점으로 가정합니다.

출력

각 테스트 케이스마다 한 줄에 x가 테스트 케이스 번호를 의미하는 "Case x:"를 출력한 후, 문자열 "Area"와 울타리로 둘러싸인 공간의 넓이를 소수로 출력한 다음 쉼표와 공백을 두고, "Perimeter"와 울타리의 둘레를 소수로 출력합니다. 정답과 출력한 소수가 절대 오차 또는 상대 오차로 $10 ^ {-9}$ 범위 내에 있다면 정답으로 간주합니다.
