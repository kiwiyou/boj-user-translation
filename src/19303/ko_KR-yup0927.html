결과값

문제

옛날 옛적에 당신에게는 멋진 양의 정수 $n$이 있었습니다.

당신은 나눗셈을 좋아했기에 $n$의 모든 양의 약수들을 구해냈습니다.

그리고 약수들의 산술 평균 $a$를 구했습니다. 놀랍게도 $a$도 정수였습니다.

시간이 지나서 당신은 약수들의 조화 평균 $h$도 구했습니다. 이 값도 정수네요!

하지만 안타깝게도 당신의 기억력이 안 좋아진 나머지 $a$와 $h$만 기억하고 $n$을 잊어버리고 말았습니다. 다만 $n$이 $10^{15}$를 초과하지 않는다는 사실만 기억하고 있죠.

좋은 추억을 되살리고자 한 당신은 $n$을 다시 알아내고 싶어졌습니다.

입력

첫째 줄에 양의 정수 $a$가 주어진다.

둘째 줄에 양의 정수 $h$가 주어진다.

산술 평균이 $a$이고 조화 평균이 $h$인 어떤 양의 정수 $n$ $(n \le 10^{15})$가 존재함이 보장된다.

출력

주어진 정보를 만족하는 양의 정수 $n$ $(n \le 10^{15})$을 출력한다.

힌트

산술 평균은 주어진 수의 합을 개수로 나눈 값이다. 예로 $1, 2, 3, 6$의 산술 평균은 $\frac{1 + 2 + 3 + 6}{4} = 3$이다.

조화 평균은 역수의 산술 평균의 역수이다. 예로 $1, 2, 3, 6$의 조화 평균은 $\big( \frac{1^{-1} + 2^{-1} + 3^{-1} + 6^{-1}}{4} \big)^{-1} = 2$이다.

예제 입력 1의 경우에 $n = 6$이라면 약수로 $1, 2, 3, 6$을 가지고, 이들의 산술 평균과 조화 평균은 각각 $3$과 $2$가 된다.
