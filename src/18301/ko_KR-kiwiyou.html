들쥐 통계학

문제

들쥐 나라의 음력 새해를 맞으며 더글러스는 마을에 있는 들쥐의 수를 조사하기로 했다. 들쥐들은 잘 숨어다니므로 모든 들쥐를 찾아다니는 것은 불가능하다. 새해 첫날에 더글러스는 n1 명의 쥐를 찾을 수 있었고, 귀에 표시를 남겼다. 이틀째에 더글러스는 n2 명의 쥐를 찾았고, 그 중 n12명이 첫날 표시되어 있었다.

더글러스는 마을에 있는 전체 들쥐의 수를 예측하려고 도움을 요청했다. 통계학 교과서에서 여러분은 다음 채프먼 추정량 공식을 찾아 사용하는 것을 제안했다.

N := ⌊(n1 + 1)(n2 + 1)/(n12 + 1) - 1⌋

⌊x⌋은 실수 x를 내림 한 값이다. 즉, x 이하의 정수 중 x에 가장 가까운 값이다.

입력

첫째 줄에 세 정수 n1, n2, n12가 주어진다.

출력

첫째 줄에 예측한 전체 들쥐의 수 N을 출력한다.

제한

  • 0 ≤ n1, n2 ≤ 10 000;
  • 0 ≤ n12 ≤ min(n1, n2).
