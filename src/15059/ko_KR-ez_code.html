힘든 선택

문제

장기 비행 때 비행사는 따뜻한 식사를 제공합니다. 보통 승무원이 비행기 통로를 따라 식사가 있는 카트를 끕니다. 카트가 당신의 줄에 도착하면, 승무원은 질문합니다: "치킨, 소고기, 파스타 중 어떤 것으로 하시겠습니까?" 당신은 선택지를 알지만, 선택할 시간은 몇 초밖에 되지 않으며 옆자리 사람이 포장을 까지 않았기 때문에 음식이 어떻게 나올지도 알 수 없는 상황입니다...

이 비행기의 승무원은 절차를 바꾸기로 했습니다. 우선 그녀는 승객에게 무슨 식사를 선호하는지 물어본 후, 비행기에 있는 식사의 수가 각 선택지만큼 충분히 있는지 확인합니다.

예를 들어, 비행기에 있는 치킨, 소고기, 파스타의 양이 각각 (80, 20, 40)인 반면, 승객이 치킨, 소고기, 파스타를 고른 수는 각각 (45, 23, 48)이라고 합시다. 이 경우, 소고기를 원한 3명과 파스타를 원한 8명은 원하는 음식을 받지 못하게 되므로 총 11명이 선호하는 음식을 못 받게 됩니다.

각 선택지의 비행기에 있는 음식의 양과 고른 손님의 수가 주어진다고 할 때, 몇 명이 자신이 원하는 음식을 받지 못하게 되는지 승무원에게 알려줄 수 있습니까?

입력

첫 줄에 각각 비행기에 있는 치킨, 소고기, 파스타의 수인 세 정수 Ca, Ba, Pa (0 ≤ Ca, Ba, Pa ≤ 100)가 주어집니다. 두 번째 줄에는 각각 치킨, 소고기, 파스타를 주문한 수인 세 정수 Cr, Br, Pr (0 ≤ Cr, Br, Pr ≤ 100)이 주어집니다.

출력

자신이 원하는 식사를 받지 못하는 손님의 수를 나타내는 정수를 한 줄에 출력합니다.
