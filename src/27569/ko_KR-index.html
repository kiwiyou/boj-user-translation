챔퍼나운 수

문제

챔퍼나운 수는 양의 정수를 차례로 이어붙인 수이다. $n$번째 챔퍼나운 수는 첫 $n$개의 양의 정수를 이어붙여 써내려간 것이다. 예를 들어, 10번째 챔퍼나운 수는 "12345678910"이다.

두 양의 정수 $n$과 $k$가 주어질 때, 첫 $n$개의 챔퍼나운 수 가운데 $k$로 나누어 떨어지는 수의 개수를 구한다.

입력

한 줄로 두 양의 정수 $n$ $(1 \le n \le 10^5)$과 $k$ $(1 \le k \le 10^9)$가 주어진다.

출력

첫 $n$개의 챔퍼나운 수 가운데 $k$로 나누어 떨어지는 수의 개수를 구한다.
