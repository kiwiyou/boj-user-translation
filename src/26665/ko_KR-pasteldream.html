피보나치 곱셈

문제

피보나치 수열은 다음과 같이 정의된다.

  • F1 = F2 = 1
  • Fk+2 = Fk+1 + Fk (단, k ≥ 1)

피보나치 진법은 양의 정수를 피보나치 수의 합으로 표현하는 방법이다. 양의 정수 X를 피보나치 진법으로 (b1, b2, …, bn)으로 표기한다. 이때, 다음과 같은 조건을 만족해야 한다.

  • X = b1 · F2 + b2 · F3 + … + bi · Fi+1 + … + bn · Fn+1
  • 각 bi는 0 또는 1이고, bn = 1이다.
  • 1이 연달아서 나올 수 없다. 즉, 1 ≤ i < n일 때 bi · bi+1 = 0이다.

인덱스에 주의하자. F1은 사용하지 않는다.

모든 양의 정수에 대해 그 정수를 피보나치 진법으로 표현하는 방법은 유일하다. 다음은 양의 정수를 피보나치 진법으로 표현한 예시이다.

  • 2: (0, 1)
  • 4: (1, 0, 1)
  • 5: (0, 0, 0, 1)
  • 20: (0, 1, 0, 1, 0, 1)

피보나치 진법으로 표현된 두 개의 양의 정수 X, Y가 주어진다. X와 Y의 곱 X · Y를 피보나치 진법으로 표현해보자.

입력

입력의 첫번째 줄에는 테스트케이스의 수 T가 주어진다. (1 ≤ T ≤ 1,000)

각 테스트케이스는 두 줄에 걸쳐서 주어진다.

테스트케이스의 첫번째 줄에는 양의 정수 X를 피보나치 진법으로 표현했을 때 n, b1, b2, …, bn의 값이 순서대로 공백으로 구분되어 주어진다.

테스트케이스의 두번째 줄에는 양의 정수 Y에 대해 마찬가지 방법으로 주어진다.

입력의 모든 n 값의 합은 1,000,000 = 106 이하 이다.

출력

각 테스트케이스마다 X와 Y의 곱 X · Y에 대해 입력과 마찬가지 방법으로 출력한다.

힌트

예제 입력 1의 첫번째 테스트케이스에서 (1, 0, 1)과 (0, 0, 0, 1)을 곱해야 한다.

(1 · F2 + 0 · F3 + 1 · F4) · (0 · F2 + 0 · F3 + 0 · F4 + 1 · F4) = (1 + 3) · (5) = 20 = F3 + F5 + F7

위의 식이 성립하므로, 곱한 결과는 (0, 1, 0, 1, 0, 1)이다.
