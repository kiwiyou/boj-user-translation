아오이와 G

문제

JOI 고등학교의 학생 아오이는 문자열 JOIG가 문자열 JOI의 끝에 문자 G를 붙인 거라는 사실을 깨달았다.

아오이는 여러 가지 뭍자열의 끝에 G를 붙였다 뗐다 하면서 놀기 시작했다.

아오이가 놀이를 하는 규칙은 다음과 같다.

  • 문자열의 맨 끝 문자가 G인 경우에는 끝에서 문자 G를 떼어낸 문자열을 생각한다. 마지막 한 문자만 떼어낸다.
  • 문자열의 맨 끝 문자가 G가 아닌 경우 끝에 문자 G를 하나 붙인 문자열을 생각한다.

길이 N인 문자열 S가 주어질 때, 아오이가 S를 보고 생각한 문자열을 출력하자.

입력

입력은 아래와 같이 주어진다.

N
S

출력

아오이가 S를 보고 생각한 문자열을 출력한다.

제한

  • 2 ≤ N ≤ 100
  • S의 길이는 N이다.
  • S는 영어 알파벳 대문자로만 이루어져 있다.
  • N은 정수이다.

문자열 JOIG의 끝 문자는 G이므로, 아오이는 문자열의 끝에서 G를 하나 뗀 문자열 JOI를 생각한다. 따라서 JOI를 출력한다.

문자열 JOI의 끝 문자는 G가 아니므로, 아오이는 문자열의 끝에서 G를 하나 붙인 문자열 JOIG를 생각한다. 따라서 JOIG를 출력한다.

문자열 EGG의 끝 문자는 G이므로, 아오이는 문자열의 끝에서 G를 하나 뗀 문자열 EG를 생각한다. 따라서 EG를 출력한다.
