Fenced In (Platinum)

문제

농부 존은 자신이 키우는 소들 중 많은 수가 광장공포증이 있다는 것을 알게 되었다. (광장공포증이란 광장과 같은 넓게 트인 공간을 두려워하는 질환이다.) 그는 소들이 편안하게 풀을 뜯어먹을 수 있도록, 수직 울타리들과 수평 울타리들을 세워서 자신의 넓은 들판을 여러 개의 작은 공간으로 나눴다.

들판은 $(0,0)$과 $(A,B)$를 꼭짓점으로 하는 직사각형 모양이다. 농부 존은 $n$개의 수직 울타리들을 서로 다른 위치 $a_1 \ldots a_n$($0 < a_i < A$)에 세웠다. 각 수직 울타리들은 $(a_i,0)$부터 $(a_i,B)$까지 이어진다. 또한, $m$개의 수평 울타리들을 서로 다른 위치 $b_1 \ldots b_m$($0 < b_i < B$)에 세웠다. 각 수평 울타리들은 $(0,b_i)$부터 $(A,b_i)$까지 이어진다. 각 수직 울타리들은 각 수평 울타리들과 교차하여 들판을 $(n+1)(m+1)$개의 영역으로 나눈다.

공교롭게도, 농부 존은 울타리들에 문을 만드는 것을 깜빡했다. 때문에 소들이 넓은 들판의 이곳저곳을 돌아다닐 수 없게 되었다. 그는 소가 인접한 영역 사이를 이동할 수 있도록 울타리들의 일부를 제거하여 이 상황을 해결하려고 한다. 그는 인접한 영역 쌍을 선택하여, 그 두 영역을 갈라놓는 울타리 부분을 제거하기를 반복하여 결국에는 소가 들판의 어느 곳으로든 이동할 수 있도록 만들 것이다.

예를 들어, 농부 존은 다음과 같이 울타리를 세울 수 있다.

+---+--+
|   |  |
+---+--+
|   |  |  
|   |  |
+---+--+

그리고 다음과 같이 울타리 일부를 제거할 수 있다.

+---+--+
|      |  
+---+  +  
|      |  
|      |
+---+--+

농부 존을 도와 그가 그의 목표를 달성하기 위해 제거해야 하는 울타리의 총 길이의 최솟값을 찾아라.

입력

첫 번째 줄에는 $A$, $B$, $n$, $m$이 차례대로 공백으로 구분되어 주어진다. ($1 \le A, B \le 1\,000\,000\,000$)

두 번째 줄부터 $n$개의 줄에 걸쳐 $a_1, \ldots ,a_n$이 하나씩 차례대로 주어진다. ($0 \le n \le 25\,000$)

이어서 $m$개의 줄에 걸쳐 $b_1,\ldots,b_m$이 하나씩 차례대로 주어진다. ($0 \leq m \leq 25\,000$)

출력

농부 존이 제거해야 하는 울타리의 길이의 최솟값을 출력하라. 답이 32-bit 정수형보다 커질 수 있으므로 64-bit 정수형을 사용해야 함에 유의하라.
